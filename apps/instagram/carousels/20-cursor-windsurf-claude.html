Windsurf = the gentler ramp.
Cascade agent asks before doing anything. Cleaner UI than Cursor. $5/mo cheaper. Best pick if you're new to AI-assisted coding and don't want the AI making moves you didn't approve.

03 · claude-code
Claude Code = pure terminal.
No GUI, no extensions, no electron. Open a terminal, type `claude`, talk to your repo. For devs who already live in tmux + vim and don't want a third VS Code instance eating their RAM.
BYO API · Opus + Sonnet · CLI

When to pick which.
Daily heavy use → Cursor 2. New to AI coding → Windsurf. Terminal native + want to script agents into CI → Claude Code. All three use the same underlying models — the difference is workflow ergonomics.
